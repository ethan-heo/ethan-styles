## color

color 속성은 프로젝트에서 사용될 기본 색상으로 구성합니다. 빨강, 초록, 파랑 등 기본 색상에서 채도를 10% 단위로 구분하여 총 10가지 색상으로 나누어 사용합니다.

- 네이밍 규칙
  - color.[COLOR_NAME].[SATURATION_NUMBER]
    - **COLOR_NAME**: 색상 이름
    - **SATURATION_NUMBER**: 채도 단위를 나타내며 숫자가 작을 수록 크며 클 수록 낮아진다.

```json
{
	"color": {
		"red": {
			"1": {
				"$type": "color",
				"$value": "#ff0000"
			}
		}
	}
}
```
