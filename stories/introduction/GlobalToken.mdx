> 글로벌 토큰을 구성하고 있는 스타일 속성에 대해 설명합니다. 글로벌 토큰에 대한 설명은 [여기서](/docs/introduction-디자인-토큰-가이드--docs#글로벌-토큰) 확인할 수 있습니다.

## color

> 프로젝트에서 사용될 기본 색상으로 구성합니다.

빨강, 초록, 파랑 등 기본 색상에서 채도를 10% 단위로 구분하여 총 10가지 색상으로 나누어 사용합니다.

### 네이밍 규칙

**color.[COLOR_NAME].[SATURATION_NUMBER]**

- **COLOR_NAME**: 색상 이름
- **SATURATION_NUMBER**: 채도 단위를 나타내며 숫자가 작을 수록 크며 클 수록 낮아진다.

```json
{
	"color": {
		"red": {
			"1": {
				"$type": "color",
				"$value": "#ff0000"
			}
		}
	}
}
```

## font-size

> 글자 크기를 정의합니다.

rem 단위로 값을 정의하여 일관된 비율의 글자 크기를 유지하게 하며 여러 형태의 디자인에서 쉽게 변경할 수 있도록 합니다.

### 네이밍 규칙

**font-size.[NUMBER]**

- **NUMBER**: 1 ~ 20 까지의 숫자로 나타내며 오름차순으로 값의 크기가 증가한다.

```json
{
	"font-size": {
		"1": {
			"$type": "dimension",
			"$value": "0.625rem"
		},
		"2": {
			"$type": "dimension",
			"$value": "0.75rem"
		},
		...
		"20": {
			"$type": "dimension",
			"$value": "3rem"
		},
	}
}
```

## font-weight

> 글자 굵기를 정의합니다.

글자 굵기는 [MDN](https://developer.mozilla.org/ko/docs/Web/CSS/font-weight#%EC%83%81%EB%8C%80%EC%A0%81_%EA%B0%80%EC%A4%91%EC%B9%98%EC%9D%98_%EC%9D%98%EB%AF%B8)에 정의된 속성값을 사용합니다.

### 네이밍 규칙

**font-weight.[WEIGHT]**

- **WEIGHT**: 가중치를 나타내며 100 단위로 100 ~ 1000 까지 나타냅니다.

```json
{
	"font-weight": {
		"100": {
			"$type": "number",
			"$value": "100"
		},
		"200": {
			"$type": "number",
			"$value": "200"
		},
		...
		"1000": {
			"$type": "number",
			"$value": "1000"
		},
	}
}
```

## line-height

> 텍스트 라인 사이 거리를 정의합니다.

거리는 배율을 사용한 상대적인 값을 사용합니다.

### 네이밍 규칙

**line-height.[NUMBER]**

- **NUMBER**: 1 ~ 20 까지의 숫자를 나타내며 오름차순으로 0.2 단위로 배율이 증가합니다.

```json
{
	"line-height": {
		"1": {
			"$type": "number",
			"$value": "1.0"
		},
		"2": {
			"$type": "number",
			"$value": "1.2"
		},
		...
		"20": {
			"$type": "number",
			"$value": "3.0"
		},
	}
}
```

## font-family

> 폰트 글씨체를 정의합니다.

기본값을 미리 정의하여 글씨체가 추가될 때 후순위로 기본값을 설정합니다.

### 네이밍 규칙

**font-family.[FONT_NAME]**

- **FONT_NAME**: 추가할 글씨체 이름을 나타냅니다.

```json
{
	"font-family": {
		"default": {
			"$type": "string",
			"$value": "sans-serif, serif"
		},
		"roboto": {
			"$type": "string",
			"$value": "Roboto, {font-family.default}"
		}
	}
}
```

## box-shadow

> 그림자 길이와 방향을 정의합니다.

그림자의 offset-x, offset-y, blur, spread 값을 정의합니다.

### 네이밍 규칙

**box-shadow.[BOX_SHADOW_PROP].[NUMBER]**

- **BOX_SHADOW_PROP**: offset-x, offset-y, blur-radius, spread-radius 속성값을 나타냅니다.
- **NUMBER**: 1 ~ 10 까지의 숫자로 나타내며 오름차순으로 2px 단위로 증가합니다.

```json
{
	"box-shadow": {
		"offset-x": {
			"1": {
				"$type": "dimension",
				"$value": "2px"
			}
		},
		"offset-y": {
			"1": {
				"$type": "dimension",
				"$value": "2px"
			}
		},
		"blur-radius": {
			"1": {
				"$type": "dimension",
				"$value": "2px"
			}
		},
		"spread-radius": {
			"1": {
				"$type": "dimension",
				"$value": "2px"
			}
		}
	}
}
```

## border-radius

> 테두리 곡선의 크기를 정의합니다.

### 네이밍 규칙

**border-radius.[NUMBER | full]**

- **NUMBER**: 1 ~ 20 까지의 숫자로 나타내며 오름차순으로 2px 단위로 증가합니다.
- **full**: 곡선의 최대 크기를 나타냅니다.

```json
{
	"border-radius": {
		"1": {
			"$type": "dimension",
			"$value": "2px"
		},
		"2": {
			"$type": "dimension",
			"$value": "4px"
		},
		...
		"full": {
			"$type": "dimension",
			"$value": "100%"
		},
	}
}
```

```json
	}
```


## breakpoint

> 반응형 디자인을 위한 breakpoint 너비를 정의합니다.

너비 기준은 [StatCounter Global Stats](https://gs.statcounter.com/screen-resolution-stats) 사이트를 참고하여 정의합니다.

### 네이밍 규칙

**breakpoint.[PLATFORM].[min, max]**

- **PLATFORM**: 플랫폼 종류를 나타내며 mobile-portrait, mobile-landscape, tablet-portrait, tablet-landscape, desktop 으로 나타냅니다.
- **min, max**: 너비의 최소, 최대 값을 나타내며 최소 하나의 값을 정의해야 합니다.

```json
{
	"responsive": {
		"mobile-portrait": {
			"max": {
				"$type": "dimension",
				"$value": "428px"
			}
		},
		"mobile-landscape": {
			"min": {
				"$type": "dimension",
				"$value": "429px"
			},
			"max": {
				"$type": "dimension",
				"$value": "768px"
			}
		},
		"tablet-portrait": {
			"min": {
				"$type": "dimension",
				"$value": "769px"
			},
			"max": {
				"$type": "dimension",
				"$value": "1024px"
			}
		},
		"tablet-landscape": {
			"min": {
				"$type": "dimension",
				"$value": "1025px"
			},
			"max": {
				"$type": "dimension",
				"$value": "1280px"
			}
		},
		"desktop": {
			"min": {
				"$type": "dimension",
				"$value": "1281px"
			}
		}
	}
}
```
